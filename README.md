# BLACK SQUAD 
### App da HACKING.RIO 
Grupo Black Squad da HACKING.RIO

## Dicas React-Native
Iniciando um projeto no react-native
```sh
$ react-native init Project
```
Iniciando o firebase no projeto
```sh
$ cd Project
$ npm install react-native-firebase
$ react-native link react-native-firebase
```
